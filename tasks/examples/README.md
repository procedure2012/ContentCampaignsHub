# Campaign Requirements Example

Each campaign folder must contain at minimum:
1. `campaign.json` - campaign metadata (name, description, target audience)
2. `content.md` - the campaign content/message

## Example campaign.json
```json
{
  "name": "example-campaign",
  "description": "An example campaign",
  "target": "general",
  "status": "draft"
}
```

## Example content.md
```markdown
# Campaign Title

Campaign body content goes here.
```
